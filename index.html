Team Skills Assessment

Welcome to the Assessment Game!

This is an interactive game designed to facilitate engaging assessment sessions, whether for warm-ups or more formal evaluations. It allows for dynamic question display, player tracking, and score management.

The goal of this game-like skill assessment is to provide a fun and engaging method for evaluation. It emphasizes key features such as interactivity, knowledge sharing, fairness, and inspiration, while promoting learning and self-assessment.

Preparing Your Questions

There are two steps to prepare your questions for the game:

  1. Create Questions: Go to the Create Questions page to create an Excel file with the required titles and columns. You can then export this Excel sheet from that page.
  2. Import Questions: Use the Assessment Importer to import the questions from the Excel file you created. From the importer, you can either use the questions directly in the game or export them as a JSON file for later use.

Running the Game

The game is hosted on Azure Static Web App. You can access it directly through the provided URL.

  1. Host Page: This is for the game referee or host to control the flow of the game, add players, load questions, and manage scores.
  2. Display Page: This page is meant to be shown to the audience or players, displaying the current question, player progress, and scores.
