VAD Team Skills Assessment

Welcome to the Assessment Game!

This is an interactive game designed to facilitate engaging assessment sessions, whether for warm-ups or more formal evaluations. It allows for dynamic question display, player tracking, and score management.

Preparing Your Questions

Questions for the game are loaded from a JSON file. You can either create this file manually or use the provided importer tool.

  1. Manual JSON: Create a questions.json file in the same directory as the game files. The structure should be an array of question objects, each with properties like question, answer, type (e.g., "warmup", "assessment", "intro-card"), points, and optionally options for multiple-choice questions.
  2. Using the Importer: For easier question preparation, use the Assessment Importer. This tool helps you convert questions from a spreadsheet format (e.g., Excel) into the required JSON structure.

Running the Game

To run the game, you need a local web server. If you're using Python, navigate to the game directory in your terminal and run python -m http.server 8000. Then, open the following links in your browser:

  1. Host Page: This is for the game referee or host to control the flow of the game, add players, load questions, and manage scores.
  2. Display Page: This page is meant to be shown to the audience or players, displaying the current question, player progress, and scores.

Functional Pages

The main interface for the game administrator. From here, you can add players, load questions, navigate through questions, show/hide answers, and manage player scores.
